    Proyecto Esteganografía

    Sistema para Ocultar Información dentro de Imagen con Cifrado Blowfish

    Sistema informático de escritorio desarrollado en C# que permite ocultar información secreta dentro de imágenes utilizando técnicas avanzadas de esteganografía combinadas con cifrado Blowfish para máxima seguridad. La aplicación permite insertar texto, documentos o cualquier tipo de información digital dentro de archivos de imagen sin alterar visiblemente la imagen original. El sistema incluye funciones de cifrado y descifrado, gestión de claves de seguridad, interfaz gráfica intuitiva y capacidad de procesar múltiples formatos de imagen. Ideal para comunicaciones seguras, protección de información confidencial y aplicaciones de seguridad informática donde la discreción es fundamental.
